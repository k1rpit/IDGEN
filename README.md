## 🛠 Описание / Description

**IDGEN** — это легковесный и полностью автономный консольный инструмент (CLI), написанный на Python. Утилита создана для мгновенной генерации реалистичных синтетических личностей и цифровых отпечатков. 

Идеально подходит для безопасного наполнения тестовых баз данных (Database Seeding), QA-тестирования, криптографических исследований и аудита безопасности.

## 💎 Особенности / Features

*   🌍 **Multi-Language Support**: Полная поддержка переключения между локализациями **Россия (RU)** и **МЕЖДУНАРОДНЫЙ (EN)** на старте скрипта.
*   🧠 **Clean Architecture**: 100%  код, и Race Conditions — стабильный вывод логов.
*   📊 **Deep Logging**: Генерирует развернутый цифровой след личности: Никнейм, ФИО, Страна, Город, Адрес, Email, Профессия, IPv4, Телефон и Автомобильный госномер.
*   📦 **Zero Mess**: Никакого мусора в системе, понятная структура и логика.

## 📦 Установка и запуск / Installation
 
Клонируй репозиторий:
```bash
git clone github.com
```

Перейди в папку проекта:
```bash
cd Identity-Generator
```

Установи необходимую библиотеку:
```bash
pip install faker --break-system-packages
```

Запусти инструмент:
```bash
python3 x.py
```
